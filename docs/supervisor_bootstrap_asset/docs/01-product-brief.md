# Product Brief

## 产品定义

这是一个非常薄的 **control-plane supervisor**，用于驱动已有 plan / workflow 的持续执行。

它不取代执行模型，不做业务实现，不做复杂编排，不做多 agent 操作系统。
它只做四类事：

- 维护运行状态
- 决定默认继续 / 重试 / 验证 / 升级 / 结束
- 调用 verifier 做确定性检查
- 记录事件与决策，保证恢复与审计

## 适用任务

### A. 预设长计划执行

例子：一个 10 阶段的改造计划，每阶段有明确产物与验证命令。

### B. 动态工作流

例子：按界面重构工作流逐屏执行，中间会因为当前屏状态选择 A / B 路径，但总体目标固定。

## 不适用场景

- 开放式研究探索
- 无 finish criteria 的 brainstorming
- 强依赖人工高频参与的任务
- 多 agent 并行协调（V1 不做）

## 成功标准

- 默认不打断用户
- 连续执行到真正完成或明确需要升级
- 完成判定由 verifier 驱动，不靠模型自述
- 支持中断恢复
